Module name
===========

This module was written to extend the functionality of Hotel/Resort/Property management to support :
    * Configure Property
    * Hotel Configuration
    * Check In, Check out
    * Manage Folio
    * Payment

Different reports are also provided, mainly for hotel statistics.

For further information, please visit:

* https://www.odoo.com/forum/help-1
